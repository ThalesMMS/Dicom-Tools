# Contract CLI (TypeScript)

CLI shim that implements the DICOM Tools contract, delegating operations to the Python backend (or another backend configured through `BACKING_CMD`).

## Structure

- **`index.ts`**: TypeScript source code
- **`index.js`**: Wrapper that calls the compiled code in `dist/index.js`
- **`dist/index.js`**: Compiled TypeScript output (generated by `npm run build`)
- **`tests/index.test.ts`**: TypeScript unit tests

## Build

```bash
npm install
npm run build
```

This compiles TypeScript into `dist/index.js`.

## Tests

```bash
npm test
npm run test:watch
npm run test:coverage
```

## Usage

The Tkinter interface uses this CLI through the adapter in `interface/adapters/js_cli.py`, which calls `node js/contract-cli/index.js`.

`index.js` now automatically uses the compiled TypeScript output, so after making changes to `index.ts`, run `npm run build` to refresh it.

## Supported Operations

- `info`: Read basic metadata
- `anonymize`: Anonymize a DICOM file
- `to_image`: Export a frame to an image (PNG/JPEG)
- `transcode`: Transcode transfer syntax
- `validate`: Validate a DICOM file
- `stats`: Pixel statistics
- `dump`: Full dataset dump
- `volume`: Build a 3D volume
- `nifti`: Export to NIfTI
- `echo`: C-ECHO to PACS
